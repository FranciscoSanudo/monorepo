          package main
import "fmt"

func main() {
	i := 7
	if i > 10 {
		fmt.Println("Greater than 10")}
	else if i > 5 {
		fmt.Println("Greater than 5")}
	else {
		fmt.Println("Less than 5")}
}

          Just look at the way this is express the way that this differs from python is that a function is declared with (or not) parameters, then a bracket is open and the variables are declared or called. Other thing that you’ll notice is that there is no ’elif’ here, instead type completely the words ’else if’ to evaluate this statement. TODO Parse commands output in go

The idea here is to have more traction when it comes to data consumption, here what I would like to do is a simple parsing for unstructured data and then encoding it in a way that can be manipulated in the way we want.

Here we are going to start with simple log parsing and then the difficulty level will be increased, but why more complicated? you might ask, that might not require any further work if simple parse is what we want. Well the answer is really straightforward there is a full advantage here because we can:

  • Filter out relevant logs from the rest

  • Take actions based on certain logs behaviour such as

    • Recommending the user to send an email
    • To count the number of events or even
    • To let the user know that there is a high possibility of FP based on XYZ.

Sky is the limit here, and now if you have the right tools you can pretty much do anything fast and simple with this wonderful language, lets start with the simple model.

package main

import (
  "fmt"
  "regexp"
)

func main() {
  // a line of log
  logsExample := `[2021-08-27T07:39:54.173Z] "GET /healthz HTTP/1.1" 200 - 0 61 225 - "111.114.195.106,10.0.0.11" "okhttp/3.12.1" "0557b0bd-4c1c-4c7a-ab7f-2120d67bee2f" "example.com" "172.16.0.1:8080"`

  // your defined log format
  logsFormat := `\[$time_stamp\] \"$http_method $request_path $_\" $response_code - $_ $_ $_ - \"$ips\" \"$_\" \"$_\" \"$_\" \"$_\"`

  // transform all the defined variable into a regex-readable named format
  regexFormat := regexp.MustCompile(`\$([\w_]*)`).ReplaceAllString(logsFormat, `(?P<$1>.*)`)

  // compile the result
  re := regexp.MustCompile(regexFormat)

  // find all the matched data from the logsExample
  matches := re.FindStringSubmatch(logsExample)

  for i, k := range re.SubexpNames() {
    // ignore the first and the $_
    if i == 0 || k == "_" {
      continue
    }

    // print the defined variable
    fmt.Printf("%-15s => %s\n", k, matches[i])
  }
}

As you can see, its easy to catch the example showed above, this is something that will be fully used when the position is the same, in other words that the logs don’t change to much, that is the basic idea of the regex shown below. Of course there are more complex and exact ways to use regex to have a better accuracy on the relevant data, but for now the idea will be to keep it simple and make the assumption that will make the same every single time we are using Go.

Now, lets take another approach, we can simply ’scan’ a document in this case a log file and search for target words or any other particular regex pattern that can have a meaning, this is another powerful approach in this case for alerting.

package main

import (
    "bufio"
    "fmt"
    "os"
    "strings"
)

func main() {
    request := 0
    f, err := os.Open("request.log")
    if err != nil {
        fmt.Print("There has been an error!: ", err)
    }
    defer f.Close()
    scanner := bufio.NewScanner(f)

    for scanner.Scan() {
        if strings.Contains(scanner.Text(), "REQUEST-A") {
            request = request + 1
        }

        if err := scanner.Err(); err != nil {
        }
        fmt.Println(request)
    }
}

TODO Web scrapper

In my opinion, this is a good approach to gain some grasp in the language, since it provides a clear image on how data manipulation can be used in real day to day applications, lets use the next piece of code to do a parsing.

The complete tutorial can be found here: Golang Web Scraper Tutorial | Oxylabs which I currently find really useful and easy to get the idea.

package main

import (
	// "encoding/csv"
	"fmt"
	// "log"
	// "os"

	"github.com/gocolly/colly"
)

func main() {
	//Scrapping code here
	c := colly.NewCollector(
		colly.AllowedDomains("books.toscrape.com"),
	)

	c.OnHTML("title", func(e *colly.HTMLElement) {
		fmt.Println(e.Text)
	})

	c.OnResponse(func(r *colly.Response) {
		fmt.Println(r.StatusCode)
	})

	c.OnRequest(func(r *colly.Request) {
		fmt.Println("Visiting", r.URL)
	})

	c.Visit("https://books.toscrape.com/")
	fmt.Println("Done")
}
